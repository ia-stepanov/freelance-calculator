# Калькулятор фрилансера

## Описание проекта
Калькулятор фрилансера — это интерактивное приложение и незаменимый помощник фрилансера для формирования стоимости заказа. Функционал разработан на основе JS.

## Функционал:
- Расчёт почасовой ставки;
- Расчёт стоимости проекта по формулам:
  - Простая формула;
  - Метод RAP;
  - Формула с коэффициентом правок.

## Стек технологий:
- HTML5 & CSS3:
  - Bootstrap;
  - Adaptive UI;
- JavaScript:
  - Стиль CamelCase.

## Языки:
- JavaScript
